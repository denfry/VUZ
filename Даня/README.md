## Структура проекта

### Web-технологии (папка `Web/`)
Лабораторные работы по HTML, CSS, JavaScript и PHP:

- **ЛР 2**: Сайт тренажерного зала с использованием цветов, изображений и гиперссылок
- **ЛР 3**: Форма декларации уплаты транспортного налога (вариант 19)
- **ЛР 4**: Подключение внешних CSS стилей, адаптивная верстка
- **ЛР 5**: Таблица степеней числа (a^i от 1 до n) с использованием JavaScript
- **ЛР 6**: Валидация формы транспортного налога с помощью JavaScript
- **ЛР 7**: Работа с массивами на PHP

### Java-приложения (папки `Lab1/` - `Lab5/`)
Лабораторные работы по программированию на Java:

- **Lab1**: Работа с массивами
- **Lab2**: Классы и объекты (Холодильник, Бонусная карта)
- **Lab3**: Калькулятор суммы
- **Lab4**: Форма нарушения ПДД
- **Lab5**: Обработка событий в формах

## Запуск

### Web-приложения
1. Для HTML/CSS/JS файлов: просто откройте `index.html` в браузере
2. Для PHP файлов: запустите локальный сервер:
   ```bash
   cd Web
   php -S localhost:8000
   ```
   Затем откройте `http://localhost:8000/lab7.php`

### Java-приложения
Скомпилируйте и запустите через командную строку:
```bash
javac LabX/FileName.java
java LabX.FileName
```
