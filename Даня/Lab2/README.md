# Лабораторная работа 2

## Задание 1: Наследование классов

### Вариант 1.19 - Холодильники

**Базовый класс:** `Refrigerator` (Холодильник)
- Поля: Производитель, цвет, высота, ширина, глубина
- Методы: вычисление объема, получение информации об объекте

**Подкласс:** `RefrigeratorWithFreezer` (Холодильник с морозильной камерой)
- Дополнительные поля: 
  - Вид камеры (класс `FreezerChamberType` с кодом и наименованием)
  - Количество камер
  - Объем морозильной камеры
- Переопределен метод получения информации

**Тестовый класс:** `RefrigeratorTest`
- Создает массив с типами элементов базового класса
- Выводит информацию о холодильниках с объемом морозильной камеры больше заданной
- Подсчитывает число холодильников и число камер в холодильниках заданного производителя

**Запуск:**
```bash
javac *.java
java RefrigeratorTest
```

## Задание 2: Интерфейсы и коллекции

### Вариант 2.19 - Ведение бонусной карты покупателя

**Интерфейс:** `BonusCardInterface`
- Определяет функциональность для работы с бонусными картами

**Класс объекта:** `BonusCard`
- Хранит информацию о карте: ID, имя владельца, баланс бонусов

**Класс-реализация:** `BonusCardManager`
- Реализует интерфейс `BonusCardInterface`
- Использует `HashMap<String, BonusCard>` для хранения карт с доступом по ключу (ID карты)
- Методы обрабатывают ошибочные ситуации:
  - Отсутствие объекта в хранилище
  - Неправильные конфигурации данных (пустые значения, отрицательные суммы, недостаточный баланс)
  - Возвращают результат выполнения операции (true/false)

**Функциональность:**
1. Открытие карты
2. Начисление бонусов от стоимости покупки
3. Списание бонусов со счета
4. Обнуление бонусов
5. Получение информации об остатке бонусов

**Тестовый класс:** `BonusCardTest`
- Тестирует все методы с различными сценариями
- Включает тесты на обработку ошибок

**Запуск:**
```bash
javac *.java
java BonusCardTest
```
