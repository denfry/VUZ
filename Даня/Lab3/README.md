# Лабораторная работа 3

## Задание: GUI приложение с вычислением суммы

### Вариант 19

**Требование:** Создать фрейм. Поместить на панели него поля ввода слагаемых и кнопку. При нажатии кнопки вычислять сумму значений из полей и помещать ее в еще одно поле.

## Описание программы

**Файл:** `SumCalculator.java`

Программа создает графический интерфейс пользователя (GUI) с использованием Java Swing, который позволяет:
- Вводить два слагаемых в текстовые поля
- Вычислять их сумму при нажатии кнопки
- Отображать результат в отдельном поле

## Компоненты интерфейса

1. **Поле ввода первого слагаемого** - для ввода первого числа
2. **Поле ввода второго слагаемого** - для ввода второго числа
3. **Кнопка "Вычислить сумму"** - для запуска вычисления
4. **Поле результата** - для отображения суммы (только для чтения)

## Функциональность

- Вычисление суммы двух чисел (поддерживаются целые и дробные числа)
- Обработка ошибок ввода (пустые поля, неверный формат чисел)
- Вывод сообщений об ошибках в диалоговых окнах
- Поддержка нажатия Enter в полях ввода для быстрого вычисления
- Форматирование результата с двумя знаками после запятой

## Запуск программы

### Компиляция:
```bash
javac SumCalculator.java
```

### Запуск:
```bash
java SumCalculator
```

## Использование

1. Введите первое слагаемое в первое поле
2. Введите второе слагаемое во второе поле
3. Нажмите кнопку "Вычислить сумму" или нажмите Enter в любом из полей
4. Результат отобразится в поле "Сумма"

## Технические детали

- Используется `JFrame` для создания главного окна
- Компоненты размещены с помощью `GridBagLayout`
- Обработка событий реализована через `ActionListener`
- Поддержка системного Look and Feel для нативного вида интерфейса
