# Лабораторная работа №6 - Каталог холодильников (JavaFX)

## Описание
JavaFX приложение для управления каталогом холодильников с табличным интерфейсом.

## Класс данных
Класс **Refrigerator** (Холодильник) содержит следующие поля:
- **Производитель** (manufacturer) - строка
- **Цвет** (color) - строка
- **Высота** (height) - число (см)
- **Ширина** (width) - число (см)
- **Глубина** (depth) - число (см)
- **Объем** - вычисляемое поле (л)

## Функциональность
1. **Просмотр** - таблица со всеми холодильниками
2. **Добавление** - добавление нового холодильника
3. **Редактирование** - изменение данных выбранного холодильника
4. **Удаление** - удаление выбранного холодильника
5. **Автоматический расчет объема** - объем рассчитывается автоматически

```bash

```

### Через IDE (Eclipse/IntelliJ):
1. Импортировать проект как Maven проект
2. Запустить класс `com.MainApp`

## Использование

### Просмотр данных
- В верхней части окна отображается таблица со всеми холодильниками
- Колонка "Объем" рассчитывается автоматически по формуле: (высота × ширина × глубина) / 1000

### Добавление холодильника
1. Заполните все поля в нижней панели редактирования
2. Нажмите кнопку **"Добавить"**
3. Новый холодильник появится в таблице

### Редактирование холодильника
1. Дважды кликните по строке в таблице - данные скопируются в поля редактирования
2. Измените нужные значения
3. Нажмите кнопку **"Изменить"**

### Удаление холодильника
1. Выберите строку в таблице
2. Нажмите кнопку **"Удалить"**
3. Подтвердите удаление

## Особенности реализации
- Использование JavaFX Properties для автоматического обновления таблицы
- ObservableList для синхронизации данных с представлением
- Валидация входных данных
- Форматирование числовых значений в таблице
- Диалоговые окна для подтверждения операций
- Автоматический расчет объема при изменении размеров
