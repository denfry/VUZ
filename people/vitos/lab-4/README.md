# Лабораторная работа 4

## Форма "Информация о сотруднике"

### Описание:
Создана форма для ввода информации о сотруднике с использованием различных компонентов Swing.

### Компоненты интерфейса:

1. **Текстовые поля (JTextField):**
   - Фамилия
   - Имя
   - Отчество
   - Оклад
   - Дата зачисления

2. **Выпадающие списки (JComboBox):**
   - Отдел (с возможностью ввода собственного значения)
   - Должность (с возможностью ввода собственного значения)

3. **Кнопки (JButton):**
   - Сохранить - сохраняет введенную информацию и выводит диалоговое окно с результатом
   - Отмена - очищает все поля формы

### Функциональность:

- **Валидация данных:** Проверка обязательных полей (Фамилия и Имя)
- **Сохранение:** Вывод информации о сотруднике в диалоговом окне
- **Очистка:** Сброс всех полей формы при нажатии "Отмена"

### Структура формы:

```
Информация о сотруднике
├── Фамилия: [текстовое поле]
├── Имя: [текстовое поле]
├── Отчество: [текстовое поле]
├── Отдел: [выпадающий список]
├── Должность: [выпадающий список]
├── Оклад: [текстовое поле]
├── Дата зачисления: [текстовое поле]
└── [Сохранить] [Отмена]
```

### Запуск:

```bash
javac EmployeeForm.java
java EmployeeForm
```

### Использование:

1. Заполните поля формы информацией о сотруднике
2. Нажмите "Сохранить" для сохранения данных
3. Нажмите "Отмена" для очистки всех полей

### Технические детали:

- Использован `GridBagLayout` для точного размещения компонентов
- Компоненты размещены в две колонки: метки слева, поля ввода справа
- Кнопки размещены внизу формы по центру
- Применен системный Look and Feel для лучшего внешнего вида
