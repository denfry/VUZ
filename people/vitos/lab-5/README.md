# Лабораторная работа 5

## Обработка событий в Java

### Описание:
Расширенная форма "Информация о сотруднике" с обработкой различных событий.

### Реализованные события:

#### 1. События фокуса (FocusEvent)
- **Интерфейс:** `FocusListener`
- **Методы:** `focusGained()`, `focusLost()`
- **Функциональность:**
  - При получении фокуса элементом устанавливается светло-зеленый фон
  - При потере фокуса восстанавливается белый фон
  - Все события фокуса логируются в область лога
- **Применено к:** всем текстовым полям и выпадающим спискам

#### 2. События движения мыши (MouseMotionEvent)
- **Интерфейс:** `MouseMotionListener`
- **Метод:** `mouseMoved()`
- **Функциональность:**
  - При перемещении курсора мыши в области формы отображаются координаты (X, Y)
  - Координаты обновляются в реальном времени в метке внизу формы
- **Применено к:** всей форме и контейнеру

#### 3. События изменения текста (DocumentEvent)
- **Интерфейс:** `DocumentListener`
- **Методы:** `insertUpdate()`, `removeUpdate()`, `changedUpdate()`
- **Функциональность:**
  - При изменении текста в поле ввода (добавление, удаление, изменение) выводится информация в лог
  - Отображается имя поля, тип изменения и длина текста
- **Применено к:** всем текстовым полям (Фамилия, Имя, Отчество, Оклад, Дата зачисления)

### Дополнительные события:

#### События окна (WindowEvent)
- **Интерфейс:** `WindowListener` (через `WindowAdapter`)
- **Методы:** `windowOpened()`, `windowClosing()`, `windowIconified()`, `windowDeiconified()`
- **Функциональность:**
  - Логирование событий открытия, закрытия, свертывания и восстановления окна

### Компоненты интерфейса:

1. **Форма ввода** (левая панель):
   - Поля для ввода информации о сотруднике
   - Кнопки "Сохранить" и "Отмена"
   - Метка с координатами мыши

2. **Лог событий** (правая панель):
   - Многострочное текстовое поле для отображения всех событий
   - Автоматическая прокрутка к последнему событию
   - Временные метки для каждого события

### Структура событий:

```
События в Java
├── FocusEvent (фокус ввода)
│   ├── focusGained() - получение фокуса
│   └── focusLost() - потеря фокуса
├── MouseMotionEvent (движение мыши)
│   └── mouseMoved() - перемещение мыши
├── DocumentEvent (изменение текста)
│   ├── insertUpdate() - добавление текста
│   ├── removeUpdate() - удаление текста
│   └── changedUpdate() - изменение текста
└── WindowEvent (события окна)
    ├── windowOpened() - открытие
    ├── windowClosing() - закрытие
    ├── windowIconified() - свертывание
    └── windowDeiconified() - восстановление
```

### Запуск:

```bash
javac EmployeeFormWithEvents.java
java EmployeeFormWithEvents
```

### Использование:

1. **Тестирование событий фокуса:**
   - Кликните по любому полю ввода
   - Обратите внимание на изменение цвета фона
   - Переключитесь на другое поле - предыдущее вернет белый фон

2. **Тестирование движения мыши:**
   - Перемещайте мышь по форме
   - Наблюдайте за изменением координат в метке внизу

3. **Тестирование изменения текста:**
   - Введите текст в любое поле
   - Удалите часть текста
   - Наблюдайте за записями в логе событий

4. **Тестирование событий окна:**
   - Сверните окно
   - Восстановите окно
   - Наблюдайте за записями в логе

### Технические детали:

- Использован шаблон делегирования для обработки событий
- Применены классы-адаптеры (`WindowAdapter`, `MouseMotionAdapter`)
- Реализованы интерфейсы слушателей (`FocusListener`, `MouseMotionListener`, `DocumentListener`)
- События логируются с временными метками
- Интерфейс разделен на две панели с помощью `JSplitPane`

### Примеры обработки событий:

**FocusEvent:**
```java
FocusListener focusListener = new FocusListener() {
    @Override
    public void focusGained(FocusEvent e) {
        component.setBackground(focusColor);
    }
    
    @Override
    public void focusLost(FocusEvent e) {
        component.setBackground(defaultColor);
    }
};
```

**MouseMotionEvent:**
```java
MouseMotionListener mouseMotionListener = new MouseMotionAdapter() {
    @Override
    public void mouseMoved(MouseEvent e) {
        int x = e.getX();
        int y = e.getY();
        // Обновление координат
    }
};
```

**DocumentEvent:**
```java
DocumentListener documentListener = new DocumentListener() {
    @Override
    public void insertUpdate(DocumentEvent e) {
        // Обработка добавления текста
    }
    
    @Override
    public void removeUpdate(DocumentEvent e) {
        // Обработка удаления текста
    }
    
    @Override
    public void changedUpdate(DocumentEvent e) {
        // Обработка изменения текста
    }

```
