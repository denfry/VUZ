# Лабораторная работа 3

## Конвертер единиц измерения длины

### Вариант 11

**Описание:**
Создан фрейм с JRadioButton для выбора единиц измерения (дюйм, фут, ярд) и двумя полями ввода. 
В первое поле вводится число в сантиметрах. При выборе переключателя значение автоматически 
пересчитывается в выбранной единице измерения и выводится во второе поле.

### Функциональность:

1. **Поле ввода** - для ввода значения в сантиметрах
2. **Переключатели единиц измерения:**
   - Дюйм (inch) - 1 дюйм = 2.54 см
   - Фут (foot) - 1 фут = 30.48 см
   - Ярд (yard) - 1 ярд = 91.44 см
3. **Поле вывода** - отображает результат конвертации (только для чтения)

### Особенности реализации:

- Автоматическая конвертация при изменении:
  - Выбора единицы измерения (JRadioButton)
  - Значения в поле ввода (обработка событий клавиатуры)
- Обработка ошибок:
  - Проверка формата введенного числа
  - Проверка на отрицательные значения
  - Визуальная индикация ошибок (красный фон)
- Форматирование результата с точностью до 6 знаков после запятой
- Использование ButtonGroup для группировки переключателей

### Компоненты интерфейса:

- `JFrame` - главное окно приложения
- `JRadioButton` - переключатели единиц измерения (3 штуки)
- `ButtonGroup` - группа переключателей
- `JTextField` - поля ввода и вывода (2 штуки)
- `JLabel` - метки и информационные подсказки
- `GridBagLayout` - менеджер компоновки для размещения компонентов

### Запуск:

```bash
javac UnitConverter.java
java UnitConverter
```

### Использование:

1. Введите значение в сантиметрах в первое поле
2. Выберите единицу измерения с помощью переключателя
3. Результат автоматически отобразится во втором поле

### Примеры:

- 2.54 см = 1 дюйм
- 30.48 см = 1 фут
- 91.44 см = 1 ярд
- 100 см = 39.370079 дюйм = 3.280840 фут = 1.093613 ярд
