# Лабораторная работа 2

## Задание 1: Наследование классов

### Вариант 1.11 - Компьютеры

**Базовый класс:** `Computer` (Компьютеры)
- Поля: Производитель, Область использования, Год выпуска
- Методы: получение информации об объекте

**Класс:** `Processor` (Процессор)
- Поля: Производитель процессора, Модель процессора
- Используется как поле в подклассе

**Подкласс:** `PersonalComputer` (Персональный компьютер)
- Дополнительные поля:
  - Процессор (класс `Processor` с производителем и моделью)
  - Модель ПК
  - Цена
  - Дата закупки
  - Установленная ОС
- Переопределен метод получения информации

**Тестовый класс:** `ComputerTest`
- Создает массив с типами элементов базового класса
- Выводит информацию о компьютерах с заданным типом процессора
- Подсчитывает число компьютеров с установленной ОС Windows

**Запуск:**
```bash
javac *.java
java ComputerTest
```

## Задание 2: Интерфейсы и коллекции

### Вариант 2.11 - Ведение постоплатного тарифного плана сотовой компании

**Интерфейс:** `PostpaidPlanInterface`
- Определяет функциональность для работы с постоплатными тарифами

**Класс объекта:** `PhoneNumber`
- Хранит информацию о номере: номер телефона, имя клиента, остаток средств

**Класс-реализация:** `PostpaidPlanManager`
- Реализует интерфейс `PostpaidPlanInterface`
- Использует `HashMap<String, PhoneNumber>` для хранения номеров с доступом по ключу (номер телефона)
- Методы обрабатывают ошибочные ситуации:
  - Отсутствие объекта в хранилище
  - Неправильные конфигурации данных (пустые значения, отрицательные суммы, недостаточный баланс, неверный формат номера)
  - Возвращают результат выполнения операции (true/false)

**Функциональность:**
1. Открытие номера
2. Зачисление средств
3. Списание средств за разговор
4. Списание средств за СМС
5. Получение информации об остатке средств
6. Получение информации о номере

**Тестовый класс:** `PostpaidPlanTest`
- Тестирует все методы с различными сценариями
- Включает тесты на обработку ошибок

**Запуск:**
```bash
javac *.java
java PostpaidPlanTest
```

## Запуск всех заданий
