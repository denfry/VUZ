## Технологии

- **Java 8+**: Объектно-ориентированное программирование, Swing GUI
- **HTML5**: Структура веб-страниц
- **CSS3**: Стилизация и адаптивная верстка
- **JavaScript (ES6+)**: Интерактивность и валидация форм
- **PHP 7.4+**: Работа с массивами на сервере

## Запуск проектов

### Java-приложения
```bash
cd [папка_студента]/Lab[X]
javac *.java
java MainClass
```

### Web-приложения
1. **HTML/CSS/JS**: Просто откройте `index.html` в браузере
2. **PHP**: Запустите локальный сервер:
   ```bash
   cd [папка_студента]/Web
   php -S localhost:8000
   ```
   Затем откройте `http://localhost:8000/lab7.php`
