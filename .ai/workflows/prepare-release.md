# Workflow: Prepare Release

## Discovery
- Confirm release scope, owner, and target modules.

## Impact Graph
- Enumerate code, docs, configs, and migrations involved.

## Risk Analysis
- Estimate blast radius and rollback complexity.

## Compatibility Verification
- Check backward compatibility and run-script stability.

## Rollback Path
- Document exact rollback command/path and data recovery steps.

## Docs Updates
- Update changelog, release notes, migration, rollback, upgrade docs.

## Test Expansion
- Execute module builds/tests and critical flow smoke checks.

## Telemetry Validation
- Confirm release monitoring points are available.

## Post-Change Governance Updates
- Record release lessons in AI learning loop.
